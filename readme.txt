== Description ==

Plugins pievieno katram rakstam "Ieteikt Draugiem" pogu.<br />
Salabots un iet kā paredzēts. Pievienota arī iespēja izvelēties kuros posts rādīt ieteikt pogu.<br />
Ja pamani kļūdu, sūti to man Twitterī ( @Methemer ).<br /><br />

English: This plugin automatically adds a "Suggest to your Friends" button from Latvian portal draugiem.lv (similiar to Facebook "Like" button).

== Installation ==
Aktivizē Pluginu un tas arī viss :) <br />
Pie "Plugins" ir arī iespēju panelis (ar vienu iespēju) "Ieteikt Draugiem", vairāk informācijas meklē tur.
Activate the Plugin & You're good to go :)
